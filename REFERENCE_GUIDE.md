# Bible Reference Study Tool

## Reference Guide

This guide is a practical companion for using the Bible Reference Study Tool. It is written for normal use: how to open passages, study them, save your work, use hymnals, manage documents, and find things again later.

---

## Quick Start

| Goal | Use This |
| --- | --- |
| Open a verse or chapter | Passage box at the top |
| Browse books and chapters | Left panel |
| Search Bible text | Left-panel search |
| Search notes, journals, maps, hymns, and documents | `Search > Search Everything` |
| Write notes for a passage | Right-panel Personal Notes |
| Create a deeper study | `Study > Passage Worksheet` |
| Gather material for a lesson or sermon | `Study > Study Binder` |
| Open hymnals and sheet music | Right panel, `Open Hymnal Reader` |
| Change colors and fonts | `Manage > Settings` |
| Open help | `Help > Open Help` or `F1` |

## 1. Main Screen

The main window is divided into three study areas.

### Left Panel

Use the left panel to:

- Choose a translation.
- Browse books and chapters.
- Search Bible text.
- View themes and thematic suggestions.
- View people connected to the current passage.
- Manage bookmarks.
- Open the Library Manager.

### Center Reader

Use the center reader to:

- Read the current chapter or passage.
- Click a verse to make it active.
- Open selected passages.
- Highlight selected text.

### Right Panel

Use the right panel to:

- View cross references.
- Open related maps.
- See people in the current passage.
- Link and open hymns.
- Read local commentary.
- Write personal notes.

### Top Menus

| Menu | What It Contains |
| --- | --- |
| Study | Dashboard, worksheet, binder, study sessions, reading plans |
| Search | Search Everything, Word Study, translation comparison, statistics |
| Explore | Timeline, cross-reference graph, presentation mode |
| Manage | Tags, exports, settings |
| Help | Help window, reference guide, developer guide |

### Status Bar

The bottom status bar shows feedback such as:

- Loaded Bible library.
- Saved note.
- Download progress.
- Import result.
- Bookmark status.
- Document conversion result.

## 2. Opening And Reading Passages

Type a passage into the passage box at the top, then choose `Open Passage`.

Examples:

```text
John 3:16
Psalm 23
Romans 8:1-4
Jn 1:1
1Corinthians 13
```

If you type a chapter without a verse, such as `Psalm 23`, the app opens verse 1 of that chapter.

### Navigation

| Action | How |
| --- | --- |
| Go back | `Back` button or `Alt+Left` |
| Go forward | `Forward` button or `Alt+Right` |
| Refresh view | `F5` |
| Select a verse | Click it in the reader |
| Open Help | `F1` |

## 3. Translations And Offline Reading

Use the translation dropdown in the left panel.

Supported sources include:

- KJV
- JPS 1917 Hebrew Bible / Tanakh
- Imported compatible JSON Bible files
- Bundled public-domain or unrestricted modules

Some copyrighted translations, such as NIV or ESV, require a compatible local file that you are allowed to use.

### Downloading For Offline Reading

KJV and JPS 1917 can fetch missing chapters and save them locally.

Open the Library Manager to download:

- Current book
- New Testament
- Old Testament / Hebrew Bible
- Torah
- Prophets
- Writings

Once a chapter is cached, the app can open it without fetching it again.

## 4. Searching

### Bible Text Search

Use the left-panel search box to search Bible text.

Search ranges may include:

- Whole Bible
- Current book
- Current chapter
- Old Testament / Hebrew Bible
- New Testament

Click a result to open that passage.

### Search Everything

Use:

```text
Search > Search Everything
```

This searches across:

- Bible text
- Personal notes
- Journal entries
- People
- Maps
- Imported documents
- Hymn titles

Use this when you remember an idea or phrase but not where it is stored.

### Word Study

Use:

```text
Search > Word Study
```

Good word-study examples:

```text
light
covenant
kingdom
faith
mercy
wisdom
```

### Translation Comparison

Use:

```text
Search > Compare Translations
```

This shows the selected verse or passage across installed translations.

## 5. Notes, Journal, And Highlights

### Personal Notes

Personal notes are attached to the selected verse or passage.

The notes editor supports:

| Action | Control |
| --- | --- |
| Save | `Save Note` |
| Clear | `Clear` |
| Undo | `Undo` or `Ctrl+Z` |
| Redo | `Redo` or `Ctrl+Y` |

Notes auto-save shortly after you stop typing.

### Journal

Use `Journal This` to create a private journal entry for the current passage.

Use the journal for:

- Reflections
- Prayers
- Personal application
- Devotional writing
- Image links

### Highlights

Select reader text and use the context menu to highlight it. Highlights are saved for the current chapter.

## 6. Study Tools

### Study Dashboard

Use:

```text
Study > Home Dashboard
```

The dashboard is a starting point for returning to recent work.

### Passage Worksheet

Use:

```text
Study > Passage Worksheet
```

The worksheet includes:

- Observation
- Interpretation
- Application
- Questions
- Prayer
- Related Hymn
- Tags

### Web Commentary

Use:

```text
Study > Save Web Commentary
```

Paste a commentary page URL and save it as a reference-based Markdown file, such as `John 3.md`, `John 3 16.md`, or `Romans.md`. The right-side Local Commentary panel will find the saved file when you open a matching passage.

Save one page at a time for personal offline study and keep the source URL with the saved note.

### Study Sessions

Use:

```text
Study > Study Sessions
```

A study session gathers passages and notes for a topic, class, sermon, or personal study.

### Study Binder

Use:

```text
Study > Study Binder
```

The binder can gather:

- Passages
- Notes
- People
- Maps
- Hymns
- Imported documents
- Exports

### Reading Plans

Use:

```text
Study > Reading Plans
```

Reading plans help track daily or planned readings.

The built-in `Chapter A Day 2026` plan is based on the CAD26 PDF schedule. Select a day and choose `Open Worksheet` to open a SOAP-style journal with the chapter text filled in and blank Scripture, Observation, Application, and Prayer areas for your response.

Concepts can include:

- Summary
- Related readings

The right panel shows related passages for the selected verse or range.

## 9. Hymnal Reader

Place PDF hymnals here:

```text
data\hymnals
```

Open the Hymnal Reader from the right panel.

The hymnal reader can:

- Browse hymns from a left menu.
- Search hymn titles and sections.
- View sheet music.
- View extracted text.
- Mark favorites.
- Show recent hymns.
- Link a hymn to the current passage.
- Add a hymn to a study session.
- Open the original PDF.

The first time a hymnal opens, the app scans it and saves an index. Later openings use the saved index unless the PDF changes.

## 10. Document Library

Open the Library Manager to convert and view documents.

Supported file types:

- PDF
- DOCX
- TXT
- Markdown
- HTML

Converted documents become searchable through Search Everything.

If a PDF is made from scanned images, it may not produce much searchable text.

## 11. Exports, Backups, And Logs

### Exports

Use:

```text
Manage > Export Study Packet
```

Exports are saved as Markdown files in the app's exports folder.

### Backups

The app creates backups before important data writes.

Backups are stored under:

```text
%LOCALAPPDATA%\BibleReferenceStudyTool\backups
```

If a JSON file is damaged, the app quarantines it instead of deleting it.

### Logs

Logs are stored under:

```text
%LOCALAPPDATA%\BibleReferenceStudyTool\logs
```

If that folder is not available, the app falls back to the project logs folder.

## 13. Troubleshooting

| Problem | Try This |
| --- | --- |
| Passage does not open | Use a simple format like `John 3:16`; check the selected translation |
| Search finds nothing | Try a shorter word or a wider search range |
| Chapter is missing | For KJV or JPS 1917, fetch/download the chapter |
| Hymnal does not show sheet music | Make sure the PDF is readable and in `data\hymnals` |
| Document has little text | The document may be scanned images |
| App reports offline mode | Local cached and imported resources still work |
| Something unexpected happened | Check logs and backups |

## 14. Suggested Workflows

### Daily Study

1. Open the app.
2. Open the passage you want to read.
3. Select a verse or range.
4. Write a personal note.
5. Open the Passage Worksheet.
6. Check cross references.
7. Review people, maps, and related hymns.
8. Add the passage to a study session if it belongs to a larger topic.

### Sermon Or Lesson Preparation

1. Create a Study Session.
2. Add the main passage.
3. Add related passages.
4. Check people, maps, and timeline.
5. Link a hymn.
6. Convert any helpful documents.
7. Use the Passage Worksheet for structure.
8. Use the Binder to gather everything.
9. Export the study packet.
10. Use Presentation Mode if teaching from the app.

### Research Workflow

1. Start with Search Everything.
2. Save promising references to a concept.
3. Compare translations.
4. Add notes and questions.
5. Use the timeline, people reference, and maps to add context.
6. Export concept notes when ready.
